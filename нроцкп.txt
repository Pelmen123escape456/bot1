	Метод
Метод- штука, которая помогает выполнять класс. Метод это переменная.
Пример:
Class- человек
Метод - еда
Метод- сон
Метод- витамины

Аргумент- переменные, которые используются в коде для нашего метода.
Пример:
Метод- еда
Аргумент- шашлык

public class Человек{
	еда(шашлык, соус тысяча островов){
		вкусно= шашлык+соус тысяча островов;
	}
}
